# Contributing to Forever Moment UI

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`.
3.  Start the development server: `npm run dev`.

## Directory Structure

- `src/features`: core business logic and UI organized by feature.
- `src/components`: shared/global UI components.
- `src/store`: global Redux store.
- `src/hooks`: global hooks.

## Code Quality

We use ESLint to enforce code quality.

- **Linting**: `npm run lint`
